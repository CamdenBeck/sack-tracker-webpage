# Cornholio Support Site

Welcome to the Cornholio Support Site! This webpage provides essential information about the Cornholio app, including how to contact us, report issues, provide feedback, and access our privacy policy.

## Project Structure

The project consists of the following files:

- **index.html**: The main HTML file for the support webpage.
- **privacy.html**: Displays the contents of the privacy policy.
- **css/styles.css**: Contains styles for the HTML files.
- **js/main.js**: Handles interactive elements on the index.html page.
- **js/privacy.js**: Loads and displays the privacy policy content.
- **privacy_policy.txt**: Contains the privacy policy text.
- **README.md**: Documentation for the project.

## Getting Started

To set up the support webpage, follow these steps:

1. Clone the repository or download the project files.
2. Open the `index.html` file in your web browser to view the support webpage.
3. Navigate through the sections to find information about the app, contact details, and methods for reporting issues or providing feedback.
